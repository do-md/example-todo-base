# Todo Base

一个基于 [Zenith](https://www.npmjs.com/package/@do-md/zenith) 状态管理库的 Todo 示例应用。

## 技术栈

- React 18
- TypeScript
- Vite
- [@do-md/zenith](https://www.npmjs.com/package/@do-md/zenith) — 状态管理
- Immer — 不可变数据更新

## 快速开始

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:3000` 即可预览。

## 构建

```bash
npm run build
npm run preview
```
